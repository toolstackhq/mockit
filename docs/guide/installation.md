# Installation

Install MockIt in any JavaScript or TypeScript project.

## npm

```bash
npm install @toolstackhq/mockit
```

## yarn

```bash
yarn add @toolstackhq/mockit
```

## pnpm

```bash
pnpm add @toolstackhq/mockit
```

## Import

```ts
import {
  MockServer,
  HttpInterceptor,
  RemoteMockServer,
  defineConfig,
  equals,
  equalsJson,
  startsWith,
  greaterThan,
} from '@toolstackhq/mockit';
```
