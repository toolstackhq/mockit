# Installation

Install MockIt in any JavaScript or TypeScript project.

## npm

```bash
npm install mockit
```

## yarn

```bash
yarn add mockit
```

## pnpm

```bash
pnpm add mockit
```

## Import

```ts
import {
  MockServer,
  HttpInterceptor,
  defineDefaults,
  equals,
  equalsJson,
  startsWith,
  greaterThan,
} from 'mockit';
```
